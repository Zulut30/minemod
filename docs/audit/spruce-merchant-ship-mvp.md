# Spruce merchant ship MVP evidence

Дата проверки: 22 июля 2026 года.

`fixtures/fabric-1.20.1-spruce-ship` является первым runnable vertical slice
корабля для Fabric 1.20.1, а не только Blockbench-моделью.

Реализовано:

- отдельный зарегистрированный `EntityType` на основе ванильной boat physics;
- управление стандартными клавишами лодки и server-authoritative movement;
- посадка игрока обычным ПКМ;
- двойное хранилище на 54 слота, открываемое Shift + ПКМ;
- NBT-сохранение груза и выпадение содержимого при разрушении;
- предмет размещения и shaped-рецепт из еловых досок, еловых брёвен и двух сундуков;
- составной клиентский рендер из block materials: ступенчатый корпус, киль,
  палуба, борта, бушприт, руль, три мачты, шесть парусов и два видимых сундука;
- локализации `en_us` и `ru_ru`.

Проверки:

- `build` проходит на Java 17, Loom 1.6.12, Loader 0.19.3 и Fabric API
  0.92.11+1.20.1;
- обязательный Fabric GameTest создаёт сущность, подтверждает boat physics,
  spruce variant, 54 слота, NBT round trip последнего слота и посадку управляющего
  пассажира;
- GameTest server завершился с `All 1 required tests passed`;
- headless client дошёл до загрузки block atlas и главного экрана; ошибки narrator
  и OpenAL относятся к отсутствующим системным библиотекам/аудиоустройству
  серверного окружения, а не к моду.

Не доказано этим срезом:

- walkable deck и multipart collision;
- отдельная физика парусов, ветра, якоря и осадки;
- несколько независимых мест пассажиров на палубе;
- игровое screenshot-evidence сущности в воде;
- баланс скорости и столкновений в реальном survival-мире.
