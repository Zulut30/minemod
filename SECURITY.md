# Security Policy

## Поддерживаемые версии

Проект находится в experimental pre-release стадии. Security fixes применяются к текущей ветке `main`; стабильные release branches пока не публикуются.

## Сообщение об уязвимости

Не публикуйте уязвимость в открытом issue. Используйте private security advisory:

<https://github.com/Zulut30/minemod/security/advisories/new>

Полезный отчёт содержит affected commit, минимальное воспроизведение, ожидаемое влияние и сведения о том, затрагивает ли проблема ModSpec validation, workspace isolation, dependency verification, MCP boundary или build runner.

Не включайте реальные API keys, access tokens, персональные данные или чужие закрытые assets.
